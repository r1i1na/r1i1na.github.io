2024 小记 - 平凡、机遇、摆烂、向前

2024-12-30T23:00:38.000Z2024-12-30T23:00:38.000Z1k3 min常州市

前置

咕咕咕咕咕咕咕咕咕咕咕咕咕咕。
也就咕了 20 天吧，不会有事的。（确信

2024，后疫情时代。虽然表面平静，实则暗流涌动。对未来的迷茫，对琐碎生活的厌倦，对生命的感悟，或平常，或深入。

又是一年过去，转眼 2025 就在眼前。
你说得对，但是明天就是 Python 年啦！

这一年，经历了很多，学到了很多，也干了很多。

高考 (Jan - Jun)

生活索然无味，尸体 cos 人类。

1月 - 6月，奋战的半年，迷茫与希望共存的半年，无数次的跌倒，无数次的爬起。
数不清的日夜，奋战于书桌前，极少的休闲时间，呆坐在电脑前，不知何所。

这几个月，技术方面基本为 0。。。这是一个内卷的时代，文化课卷不过别人，只有被淘汰（大概也是暂时的），那有心思往别的地方放呢？

三个字：人麻了。这6个月，很快就过去了。整个人犹如机器一般，没啥感情，有的只是动笔。

5月8日，学校组织研学，难得的放松：

6月9日，高考落下帷幕，朋友各奔东西，十二年寒窗就此画上句号，新的旅程即将开始。

考完去新疆玩了一趟，风景还是很漂亮的：

Jul

众所周知，高考完并没有什么作业。（我例外）

因为下定决心去国外读大学，因此整个 7 月我都在备考 IELTS（头痛）。

7月23日，很顺利地考完了。总分 6.5，预料之中，不算太好也不算太差。

Aug

玩机之路

180 收了台小米 MIX 2s 8+256 顶配版本，电源排线有点问题，无限重启，被卖家贱卖了。

几块换了根电源排线（笑），完美复活！

???

赛博华佗啊!

前前后后刷了一大堆系统，还是很好玩的ww：

也不是一切顺利吧，有的时候就是会发电（）：

Sep

非常平凡的一个月，主要浅学了一下C++, Python, js等各种 Programming Language。

还有人类语言: English

Oct

澳大利亚

学校10月8日开学，但是为了适应，实际上10月1日就去了。

全新的 environment，全新的一切，不经让我深感好奇，这差不多也是技术进步约为 0 一个月。

来到这，真的就变成 npc 了。（悲

这种环境下，见人间烟火，感人世真情，对于 npc 来说已经很高兴了。

Nov

全年技术进步最大的的一月。

11月5日，我重开了2022年就已经尝试过的博客。使用 Hexo 框架，以及 Butterfly 主题。（也就是本站啦）

具体的博客小故事可见：
本站小记

Github 原来的账号不知道为什么掉了（），创了个新的，把原来的仓库都弄了回来。

前后陆陆续续换了几个主题，通过魔改也变相学习了css, js, html等。

剩下的便是文化科目的学习，学习，还是学习。。。

Dec

一个学期进入尾声，很顺利地结束了 Mid-term test ，目前 GPA 在 8.2。

继续在悉尼刷探索度，详情可见：
周末杂记 (12.7-8)

回国

回国之后，因为搬家的缘故，我花了一整天的时间重新整理桌子。（要命）

接下来就是日常继续对各种Programming Languages的学习了，一天天的重复着，无大事发生。

哦还有一款游戏：《彼方的她-Aliya》，虽然不到一杯奶茶钱，却足以让人产生共鸣。（俗称：后劲太大）

旅途远远没有结束…

2025，新的开始。

追求并不是很高，开心过完每一天即可。

一点小计划：

  • 做一点自己感兴趣的项目
  • 下学期的 GPA 不拉到 6.9 以下
  • 掌握C++, Python, Swift, js
  • 去一些好玩的地方
  • 不要挂科
  • 还有很多，就不一一列举了，或者说，到时候再说吧
